HTML HSL and HSLA Colors

HSL stands for hue, saturation, and lightness. HSLA color value are an extension of HSL with an Alpha channel (opacity).

HSL Color Values

In HTML, a color can be specified using hue, saturation, and lightness (HSL) in the form :

hsl (hue, saturation, lightness)

Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.

saturation is a percentage value. 0% means a shade of gray, and 100% value is the full color.

Lightness is also a percentage value. 0% is black, and 100% is white.

Example is given below :

hsl(0,100%,50%)

hsl(45,100,50%)

hsl(260,100%,50%)

Saturation

Saturation can be described as the intensity of a color.

100% is pure color, no shades of gray.

50% is 50% gray, but we can still see the color. in general, 50% of saturation is the best setting to make the hue color become solid.

0% is completely gray; we can no longer see the color.

Example is given below :

hsl(0, 100%, 50%) which is it will make a pure color

hsl(0, 0%, 50%) which mean it will make the color become weak(fully gray).

Lightness

The lightness of a color can be described as how much light you want to give the color, where 0% means no light (black), 50% means 50% light (neither dark nor light), and 100% means full brightness (white). Example is given below :

hsl(0,100%,0%) which will make the color dark (black).

hsl(0,100%,100%) which will make the color become full of brightness (white).

Shades of Gray

Shades of gray are often defined by setting the hue and saturation to 0, and adjusting the lightness from 0% to 100% to get darker / lighter shades. why? setting hue become 0 to make it easy to memorize, like just make the hue and saturation same value. Because the primary value to make the color become gray is set the saturation become 0 (completely gray, no matter what value does the hue has). Example is given below :

hsl(0,0%,50%)

hsl(232,0%,50%)

From the sample background color above, no matter what value does the hue has, it will make the color become gray. moreover, they are typical.

hsl(0,0%,25%)

hsl(0,0%,75%)

From the sample background-color above, the gray color of background color depends on the lightness of HSL. the lower the value, it will become darker and vice versa.

HSLA Color Values

HSLA color values are an extension of HSL color values, with an Alpha channel - which specifies the opacity for a color.

An HSLA color value is specified with :

hsla (hue, saturation, lightness, alpha)

The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all). The alpha channel make the color have transparency, make it more flexible to custom the color.Example is given below :

hsl(0, 100%, 50%)

hsla(0,100%,50%,1.0)

From the sample given above, to make the color typical with basic HSL, we can set the value of alpha 1.0. This alpha make the color more flexible to custom. Example is given below :

hsl(0, 100%, 50%)

hsla(0,100%,50%,0.5)
