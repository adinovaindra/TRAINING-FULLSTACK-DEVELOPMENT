HTML Colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, and HSLA values.

Color Names

In HTML, a color can be specified by using a color name like Tomato, Orange, MediumSeaGreen, and so on. There are 140 satndard color names in HTML.

Color Values

In HTML, colors can also be specified using RGB values, HEX values, HSL values, RGBA values, and HSLA values.

The following three <div> elements have their background colors set with RGB, HEX, and HSL values :

rgb(255, 99, 71)

#ff6347

hsl(9, 100%, 64%)

The following 2 <div> elements have their background color set with RGBA and HSLA values, which add an Alpha channel to the color (here we have 50% transparency)

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)
