HTML Image Maps

With HTML image maps, you can create clickable areas on an image.

Image Maps

The HTML <maps> tag defines an image map. An image map is an image with clickable areas. The areas are defined with one or more <area> tags.

How Does it Work?

The idea behind an image map is that you should be able to perform different actions depending on where in the image you click.

To create an image map you need an image, and some HTML code that describes the clickable areas.

The Image

The image is inserted using the <image> tag. The only difference from other images is that you must add a usemap attribute like this :

<img src="sourceoftheimage" alt="description of the image" usemap="#mapname">.

Tip: We can use any image as an image map!

Create Image Map

Then, add a <map> element.

The <map> element is used to create an image map, and is linked to the image by using the required name attribute like this :

<map name="mapname">

The name attribute must have the same value as the <img>'s usemap attribute.

The Areas

Then, add the clickable areas.

A clickable area is defined using an <area> element.

Shape

We must define the shape of the clickable area, and we can choose one of these values :

  • rect - defines a rectangular region
  • circle - defines a circular region
  • poly - defines a polygonal region
  • default - defines the entire region

We must also define some coordinates to be able to place the clickable area onto the image.

Shape=rect

The coordinates for shape="rect" copme in pairs, one for the x-axis and one for y-axis.

So, the as example the coordinates 34,44 is located 34 pixels from the left margin and 44 pixels from the top.

The other pairs as example 270,350. So it means 270 pixels from left margin and 350 pixels from top.

Now we have enough data to create a clickable rectangular area.

Shape=circle

To add a circle area, first locate the coordinates of the center of the circle, as example 337,300.

Then specify the radius of the circle, example : 44 pixels.

Now we have enough data to create a clickable circular area.

Shape=poly

The shape="poly" contains several coordinate points, which creates a shape formed with straight lines (a polygon).

This can be used to create any shape, like maybe a croissant shape!
