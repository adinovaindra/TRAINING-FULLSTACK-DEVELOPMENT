HTML Background Images

A background Image can be specified for almost any HTML element.

We can also specify the background image in the style element, in the head section.

Background Image on a Page

If we want the entire page to have a background image, we must specify the background image on the <body> element.

Background Repeat

To avoid the background image from repeating itself, set the background-repeat property to no-repeat.

Background Cover

If we want the background image to cover the entire element, we can set the background-size property to cover.
