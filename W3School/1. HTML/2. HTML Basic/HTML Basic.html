HTML Basic

HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPEhtml>. The HTML document itself begins with <html> and ends with </html>. And the visible part of the HTML document is between <body> and </body>.

<!DOCTYPE>

It helps us to display web correctly. it declerates HTML5.
