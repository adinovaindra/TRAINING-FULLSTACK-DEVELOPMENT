HTML Elements

The HTML element is everything from the start tag to the end tag, but some HTML elements have no content like <br> tag. It used to add line break of HTML pages. This element is called empty elements. Empty elements don't have an end tag.

Nested HTML Elements

HTML elements can be nested ( this means that elements can contain other elements).
All HTML documents consist of nesasted HTML elements. The sample of nested elements is what we are doing now. We use <p> and <h1> tag inside <body> tag.

HTML is Not Case Sensitive

HTML tags are not case sensitive : <P> means the same as <p>.
The HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML, and demands lowercase for stricter document types like XHTML.
