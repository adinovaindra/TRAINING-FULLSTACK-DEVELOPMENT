HTML <b> and <strong> Tag

The HTML <b> and <strong> tag both visualize the text become bold. The only difference is <strong> tag is semantic, which means its good for screen reader and SEO.

HTML <i> and <em> Tag

Same with <b> and <strong> tag, <i> and <em> tag both visualize the text become italic. The only difference is <i> tag doesn't semantic while <em> does.

HTML <small> Element

The <small> element defines smaller text. It is usually use for copyright.

HTML <mark> Element

The HTML <mark> element defines text that should be marked or highlighted. It will mark the text with yellow background.

HTML <del> Element

HTML <del> element defines text that has been deleted from a document. Browsers will usually strike a line throug deleted text like this.

HTML <ins> Element

The HTML <ins> element defines a text that has been inserted into a document. Browsers will usually underline the text like this.

HTML <sub> Element

The HTML <sub> element defines subscript text. It can be used for chemical fromula like this : H2o.

HTML <sup> Element

The HTML <sup> element defines superscript text. It can be used for footnotes or square root like this : 42 = 16.
