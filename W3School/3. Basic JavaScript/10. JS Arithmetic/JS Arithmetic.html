JavaScript Arithmetic Operators

Arithmetic operators perform arithmetic on numbers (literals or variables). There are :

  • + (Addition)
  • - (Substraction)
  • * (Multiplication)
  • ** (Exponentiation)
  • / (Division)
  • % (Modulus / Remainder)
  • ++ (Increment)
  • -- (Decrement)
